# ACBIB (Assetto Corsa Basic Information Board)

Assetto Corsa App that shows all relevant car and race data:
- gear
- rpm
- shift indicators
- speed
- fuel
- tyres (temperature, pressure, wear)
- position
- lap
- current lap
- best lap
- lap delta
